# Admin Login

## Goal

- Provide a simple, trustworthy entry to the protected admin surface.

## Audience

- authorized Afribit administrators only

## Live-Source Content Summary

- Live admin entry is Google-auth based and explicitly restricted to authorized administrators.

## Section Order

- simple hero or centered login card
- auth explanation
- sign-in action
- support link

## CTA Inventory

- Sign in with Google
- support contact if blocked

## Asset References

- logo assets only

## Component Candidates

- shadcn card and button primitives

## Motion Notes

- minimal

## Accessibility Notes

- auth errors must be announced clearly and without layout confusion

## Acceptance Criteria

- stays limited to login purpose and does not drift into full admin portal redesign
